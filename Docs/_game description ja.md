YUME ROOF は、日本の雑居ビルの屋上を自由にカスタマイズして、

自分だけの居心地のいい場所を作る、サンドボックス型スローライフゲームです。

マネジメントも、戦闘も、決められた遊び方もありません。

あるのは、屋上と、時間と、少しの猫だけ。


**〜〜 建築ゲームなのにグリッドなし 〜〜**

家具や植物、建物のパーツは、

グリッドに縛られず、直感的にどこへでも配置できます。

ベンチをずらしたり、鉢植えを並べたり、

室外機の隣に小さなテーブルを置いたり。

少しの違いが、屋上の表情をやさしく変えていきます。

**〜〜 ゆったりとした屋上の時間 〜〜**

朝日が差し込み、

風に植物が揺れ、

ときどき猫がふらりと現れます。

時間帯を変えれば、

夕焼けの屋上にも、夜の静かな空間にもなります。

ただ眺めているだけでも、心が少し軽くなる場所です。


**〜〜 つくるほど、あなたらしく 〜〜**

このゲームに、失敗や行き止まりはありません。

思いついたらすぐに作り直せて、

どんな配置でも、どんな色でも、

それはあなたの「屋上」になります。

完璧なデザインも、散らかった屋上も、

どちらもあなたの物語です。
