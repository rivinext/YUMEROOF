YUME ROOF 是一款沙盒式慢生活游戏，可以自由定制日本杂居楼的屋顶，

打造只属于自己的舒适空间。

没有经营，没有战斗，也没有固定的玩法。

这里只有屋顶、时间，以及几只猫。


**〜〜 建筑游戏却没有网格 〜〜**

家具、植物和建筑部件

不受网格限制，可以凭直觉摆放到任何地方。

挪动长椅、摆放花盆，

在室外机旁放一张小桌子。

一点点变化，都会温柔地改变屋顶的表情。

**〜〜 慢悠悠的屋顶时光 〜〜**

晨光洒落，

风中植物轻轻摇曳，

有时猫会晃悠悠地出现。

改变时间段，

就能来到夕阳下的屋顶，也能置身夜晚的静谧空间。

哪怕只是静静地看着，心也会稍微轻松一点。


**〜〜 越打造，越像你 〜〜**

这款游戏没有失败，也没有死胡同。

想到什么就能立刻重做，

不管怎样摆放、不管什么颜色，

那都会成为你的“屋顶”。

完美的设计也好，凌乱的屋顶也好，

它们都是你的故事。
