YUME ROOF 是一款沙盒式慢生活遊戲，可以自由客製化日本雜居大樓的屋頂，

打造只屬於自己的舒適空間。

沒有經營，沒有戰鬥，也沒有固定的玩法。

只有屋頂、時間，以及幾隻貓。


**〜〜 建築遊戲卻沒有網格 〜〜**

家具、植物與建築部件

不受網格限制，可以憑直覺擺放到任何地方。

挪動長椅、擺放盆栽，

在室外機旁放一張小桌子。

一點點差異，都會溫柔地改變屋頂的表情。

**〜〜 悠閒的屋頂時光 〜〜**

晨光灑落，

風中植物輕輕搖曳，

有時貓會晃悠悠地出現。

改變時間帶，

就能來到夕陽下的屋頂，也能置身夜晚的靜謐空間。

就算只是靜靜地看著，心也會稍微輕鬆一點。


**〜〜 越打造，越像你 〜〜**

這款遊戲沒有失敗，也沒有死胡同。

想到什麼就能立刻重做，

不管怎麼擺放、不管什麼顏色，

那都會成為你的「屋頂」。

完美的設計也好，凌亂的屋頂也好，

它們都是你的故事。
