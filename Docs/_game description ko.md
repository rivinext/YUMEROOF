YUME ROOF는 일본의 잡거(혼합) 건물 옥상을 자유롭게 커스터마이즈해

나만의 아늑한 공간을 만드는 샌드박스형 슬로 라이프 게임입니다.

경영도, 전투도, 정해진 플레이 방식도 없습니다.

있는 것은 옥상과 시간, 그리고 몇 마리의 고양이뿐입니다.


**〜〜 건축 게임인데 그리드가 없다 〜〜**

가구와 식물, 건축 파츠는

그리드에 묶이지 않고 직관적으로 어디에나 배치할 수 있습니다.

벤치를 옮기거나, 화분을 나란히 놓거나,

실외기 옆에 작은 테이블을 두는 것처럼요.

작은 차이가 옥상의 표정을 부드럽게 바꿉니다.

**〜〜 느긋한 옥상의 시간 〜〜**

아침 햇살이 비추고,

바람에 식물이 흔들리며,

가끔 고양이가 불쑥 나타나기도 합니다.

시간대를 바꾸면

석양의 옥상도, 밤의 고요한 공간도 됩니다.

그저 바라보기만 해도 마음이 조금 가벼워지는 곳입니다.


**〜〜 만들수록, 나다워진다 〜〜**

이 게임에는 실패도 막다른 길도 없습니다.

생각이 떠오르면 바로 다시 만들 수 있고,

어떤 배치든 어떤 색이든,

그것은 당신의 “옥상”이 됩니다.

완벽한 디자인도, 어질러진 옥상도,

모두 당신의 이야기입니다.
