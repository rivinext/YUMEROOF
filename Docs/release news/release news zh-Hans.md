YUME ROOF 是一款慢节奏生活沙盒游戏。
你可以在日本城市楼顶自由布置，打造只属于你的舒适角落。

没有经营压力，没有战斗，也没有“正确玩法”。
有的只是屋顶、流动的时间，以及偶尔来访的猫咪。

当日常让你疲惫时，不妨来到城市上空安静片刻。
从清晨到夜晚，按自己的节奏整理空间，享受“只是待着”的时光。

支持语言：
English / 日本語 / 简体中文 / 繁體中文 / Français / Deutsch / Italiano / Português / Español / 한국어 / Русский
